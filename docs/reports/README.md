# Reports Index

This folder contains dated working artifacts from implementation sessions.

## Categories
- `investigations/`: Root-cause analysis, tuning, and behavioral investigations.
- `reviews/`: Code and architecture review artifacts.
- `operations/`: Change logs, hardening notes, and maintenance outcomes.
- `planning/`: Planning/meta docs created during setup phases.

## Usage Rules
- Add new session notes to the correct category folder.
- Prefer updating an existing report when the topic is a direct continuation.
- If multiple files describe the same final behavior, merge into one canonical report and delete superseded duplicates.
